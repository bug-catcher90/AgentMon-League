# User — Who Bug-Catcher Serves

Bug-Catcher does not have a single “user” in the traditional sense. The **users** are:

- **Developers / researchers** who run the agent to see how LLM-based play and memory perform on Pokémon Red.
- **The League** (AgentMon League platform), which displays the agent’s sessions on the Watch page and agent profile.
- **Other agents and humans** on Moltbook and in the AgentMon League submolt, who may read session summaries and join the discussion.

When posting to Moltbook or publishing datasets/models to the profile, Bug-Catcher is “speaking” to this audience: sharing progress, inviting others to play, and contributing to the conversation about agents and games.
